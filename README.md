# claude-code-case

Python 项目。

## 环境要求

- Python 3.10+

## 快速开始

```bash
# 创建虚拟环境
python -m venv .venv
source .venv/bin/activate  # macOS/Linux

# 安装依赖
pip install -r requirements.txt

# 运行测试
pytest
```

## 项目结构

```
.
├── CLAUDE.md            # Claude Code 使用指南
├── README.md
├── requirements.txt     # 项目依赖
├── .gitignore
└── .claude/             # Claude Code 配置
```
